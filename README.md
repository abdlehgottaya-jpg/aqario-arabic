# 🎮 اقاريو العربية - Aqario Arabic

لعبة ويب تفاعلية متعددة اللاعبين، مستوحاة من لعبة Agar.io الشهيرة!

## 📋 المميزات

✨ **لعبة تفاعلية** - تحكم في خليتك واأكل الكائنات الأخرى
🎯 **أسلوب لعب متنوع** - اختر بين سيرفرات مختلفة
🌈 **رسومات جميلة** - واجهة مستخدم عربية حديثة
⚡ **أداء سريع** - مستخدمة Canvas للرسومات

## 🕹️ طريقة اللعب

1. **انقر على زر "العب"** لبدء اللعبة
2. **حرك الفأرة** للتحكم في خليتك
3. **اأكل الكائنات الملونة** لتكبير خليتك وزيادة النقاط
4. **تجنب الفيروسات الخضراء** - لأنها ستفجرك!

### المفاتيح
- `F` - تغذية (Feed)
- `G` - انقسام (Split 4)
- `H` - انقسام متعدد (Split 16)
- `J` - انقسام كبير (Split 32)
- `K` - انقسام ضخم (Split 64)
